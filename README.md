# Projeto de Cifragem em Python

Este projeto demonstra **três métodos simples de cifragem** desenvolvidos em Python:

- **Cifragem por inversão**
- **Cifragem por deslocamento (Cifra de César)**
- **Cifragem por substituição (Base64)**

---

## Como usar

1. **Edite o arquivo `meutexto.txt`**  
   Escreva nele o texto que deseja cifrar.  
   (O arquivo já está incluído neste repositório.)

2. **Execute o programa:**
   ```bash
   python3 cifragem.py
