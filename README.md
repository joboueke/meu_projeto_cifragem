# Projeto de Cifragem em Python

Este projeto demonstra três tipos simples de cifragem:
- **Cifragem por inversão**
- **Cifragem por deslocamento (Cifra de César)**
- **Cifragem por substituição (Base64)**

## Como usar

1. Coloque um arquivo chamado `meutexto.txt` na mesma pasta do script `cifragem.py`.
2. Execute o programa:
   ```bash
   python3 cifragem.py
